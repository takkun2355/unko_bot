    便利系
        weather
            ^^weather <都道府県名>
            例: ^^weather 名古屋県県
            説明: 指定した都道府県の天気を取得します。

            ^^weather Details <都道府県名>
            例: ^^weather Details 名古屋県
            説明: 指定した都道府県の詳細な気象情報を取得します。

        roulette
            ^^roulette [モード] <候補1> <候補2> ...
            例: ^^roulette 牛丼 カレー ラーメン
            説明: 1回ルーレットを実行します。

            ^^roulette [モード] loop <候補1> <候補2> ... <回数> 
            例: ^^roulette loop 牛丼 カレー ラーメン 3
            説明: 複数回ルーレットを実行します。

            モードオプション（順不同で指定可能）:
                fast         : 即結果表示モード（演出なし）
                topN<number> : 上位N位まで表示（例: topN3, topN5）
                countdown    : 回転前に3秒カウントダウン演出
                duplicate    : 候補の重複ありモード（同じ候補が複数回当選可能）
                gif          : 回転中にGIF演出（テキスト上で表現）
                background   : 背景画像ランダム適用（テキスト上で表現）
                winner       : 勝者エフェクト（✨表示）
                sound        : 音声通知（VC非対応、テキストで表示）
                history      : 履歴保存（過去ルーレット結果に追加）
                time:時間:分  : 指定時刻にルーレットを自動実行

            ^^roulettehistory
            説明: 過去10件のルーレット結果を確認します。

        translate
            ^^translate
            例: ^^^translate <翻訳後の言語> <文>
            説明: 翻訳する

            ^^languages
            説明: 言語を表示

        URL
            ^^shorten <URL>
            説明: URLを短縮

            ^^expand <URL>
            説明: 短縮URLを展開

        ^^serverstatus
        説明: サーバーの人数(botと人間別＋合計数)

        Icons
            ^^icon

            説明: 特定のユーザーのプロフィール画像を取得

            ^^servericon
            例: ^^servericon [招待リンク](https://discord.gg/xxxxxxx)
            説明: 特定のユーザーのプロフィール画像を取得
















管理系
    管理者権限
        files
            ^^files <オプション>
            例: ^^files
            説明: unkoman_bot 内のトップ層

            例: ^^files venv
            説明: venv 内のファイル表示

            例: ^^files .txt
            説明: .txt ファイルだけ表示

            例: ^^files address C
            説明: Cドライブから unkoman_bot 以下をフルパス表示

        spam
            例: ^^spam @ユーザー名 [回数] [遅延(秒)]
            説明: 指定したユーザーに繰り返しメンションを送信します。
    
    オーナー権限
        ^^restart
        説明: botを再起動します

        ^^stop
        説明: botを停止します
